Vocabulary Profile for EPUB 3 Package Documents

This RDFa 1.1 Profile document defines a default vocabulary and a set of vocabulary prefix mappings for use in EPUBÂ® 3 Package Documents.

Default Vocabulary

This profile defines the EPUB 3 Package Metadata Vocabulary as its default vocabulary.

All unprefixed CURIEs must resolve to the URI: http://www.idpf.org/epub/vocab/package/#.

Additional Vocabularies

This profile also defines prefixes for the following vocabularies:

  • Dublin Core DCTERMS Metadata

    The dcterms prefix must be used for the vocabulary URI http://purl.org/dc/terms/
  • EPUB 3 Media Overlays Metadata Vocabulary

    The media prefix must be used for the vocabulary URI http://www.idpf.org/epub/vocab/overlays/#
  • W3C XML Schema Datatypes

    The xsd prefix must be used when referencing datatypes defined in the http://www.w3.org/2001/XMLSchema namespace.
