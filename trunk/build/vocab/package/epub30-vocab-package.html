About this vocabulary

This vocabulary, a part of EPUB 3.0, defines a set of metadata properties for use in [../../spec/epub30-publications.html#sec-package-documents Package Documents].

Package-level Metadata

Publication Metadata

content-version

The version number of the publication.

source-identifier

Indicates which identifier property represents the source publication identifier.

Extended metadata properties

alternate-script

Indicates an alternate representation of the associated property in a language and/or script identified by the xml:lang attribute.

display-seq

Indicates the sequential position in which to display the current property relative to similarly-identified properties for the publication.

event

Indicates the reason for an update to a publication, as identified by the inclusion of a dcterms:date property (e.g. a change to the publication metadata).

file-as

A normalized form of the associated property for machine processing purposes.

role

Indicates the role a contributor played in the creation of the content or production of the publication (values should be drawn from ONIX 3.0 Code List 17).

scheme

Indicates the scheme the associated property value conforms to (values should be drawn from ONIX 3.0 Code List 27).

sort-order

Indicates the ordinal position in which to sort the current property relative to similarly-identified properties for the publication.

Metadata Authority properties

authority

The unique identifier for the metadata authority that issued an identifier for the publication.

date-ref

Indicates a reference to a dcterms:date for the associated property.

meta-auth

The name of a metadata authority that issued an identifier for the publication

signature

Identifies an associated signature for the metadata authority available in the signatures.xml file.

Manifest properties

cover-image

Indicates that the associated resource contains a representation of the cover art for the publication.

mathml

Indicates that the associated resource contains one or more instances of MathML markup.

online-resource

Indicates that the associated resource contains references to additional resource that are not local to the publication.

page-progression-direction-ltr

Indicates that the page progression of the associated resource is from left to right.

page-progression-direction-rtl

Indicates that the page progression of the associated resource is from right to left.

scripted

Indicates that the associated resource contains scripted components.

svg

Indicates that the associated resource contains one or more instances of SVG markup.
