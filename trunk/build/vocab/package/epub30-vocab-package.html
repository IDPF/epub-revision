About this vocabulary

This vocabulary, a part of EPUB 3.0, defines a set of metadata properties for use in [../../spec/epub30-publications.html#sec-package-documents Package Documents].

Package-level Metadata

content-version

The version number of the package.

publication-type

Describes the nature of a specialized type of EPUB Package.

source-identifier

References the identifier property that is the source of the NCX pageList.

page-progression-direction

Indicates the direction in which content flows.

Ancillary metadata properties

alternate-script

Provides an alternate expression of the associated property value in a language and script identified by the {xml:lang} attribute.

date-ref

Relates a metadata property to a {dcterms:date} property. !Why was this under metadata authority props? Is it global?^

display-seq

Indicates the order in which to display the current property relative to identical metadata properties.

event

Describes a {dcterms:date} property.

file-as

The normalized form of the associated property for sorting.

role

Describes the nature of work performed by a creator or contributor.

scheme

Identifies the formal system used to express {dcterms:identifier} or {dcterms:subject} metadata.

sort-order

Indicates the order in which to sort the associated {dcterms:title} metadata property relative to other {dcterms:title} properties with the same {title-type} property.

title-type

Indicates the nomenclature of a {dcterms:title}.

Metadata Authority properties

authority

References the metadata authority ({meta-auth}) responsible for an instance of package metadata.

meta-auth

The name of a party or authority responsible for the package metadata.

signature

Relates a metadata authority ({meta-auth}) responsible for creating or updating package metadata to an XML Signature contained in the Signatures.xml file.

Manifest properties

cover-image

Identifies a resource as the cover image for the publication.

mathml

Indicates that the associated resource contains one or more instances of MathML markup.

online-resources

Indicates that the associated resource references publication resources that are located online.

scripted

Indicates that the associated resource contains scripted content.!Should we define scripted content here or at a higher level location.^

svg

Indicates that the associated resource contains one or more instances of SVG markup.

Metadata record types

onix-record

Indicates the referenced resource is an ONIX record.

mods-record

Indicates the referenced resource is a MODS record.

xmp-record

Indicates the referenced resource is an XMP record.

marc21xml-record

Indicates the referenced resource is a MARC21 record.
