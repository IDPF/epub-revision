About this vocabulary

This vocabulary, a part of EPUB 3.0, defines a set of metadata properties for describing [../../spec/epub30-mediaoverlays.html EPUB3 Media Overlays].

Package Metadata

Properties for inclusion in EPUB3 Package Documents

duration

Clock value expressed in hh:mm:ss.s or as a single unit of h (hours), min (minutes), s (seconds), or ms (milliseconds). A subset of SMIL Wallclock-sync values

narrator

String containing the name of the narrator
