Well-known extension identifiers for EPUB switch

This document lists well-known extension identifiers for use with the EPUB switch element.

This document is informative and not tied to a particular version of the EPUB specification. It is updated periodically as new identifiers are submitted by the EPUB community.

Document last updated: 2011-05-05

MathML

The MathML namespace URI http://www.w3.org/1998/Math/MathML is a well-known identifier for MathML content.
