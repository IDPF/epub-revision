Vocabulary Profile for EPUB3 Content

This RDFa 1.1 Profile document defines a default vocabulary and a set of vocabulary prefix mappings for use in EPUB3 Content Documents and associated resource types.

Default Vocabulary

This profile defines the EPUB3 Structural Semantics Vocabulary as its default vocabulary.

All unprefixed CURIEs must resolve to the URI: http://www.idpf.org/epub/30/vocab/structure/#
