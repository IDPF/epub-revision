EPUB Publications Reserved Prefixes

Overview

This document defines the set of prefixes that Reading Systems are required to recognize when used in EPUBÂ® Package Document metadata.

For more information about prefixes, including how to declare and use them in the Package Document, please refer to Vocabulary Association Mechanisms in the Publications specification.

Revision Policy

This document is subject to change at any time. The prefixes defined herein will never be removed, but may be deprecated.

Requests for additions, modifications and clarifications must be made through the EPUB 3 Revision Tracker.

This following prefixes are defined for use in package metadata.
